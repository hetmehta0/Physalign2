# PhysAlign

PhysAlign is an innovative rehabilitation platform that uses real-time body tracking technology to help users perform exercises correctly at home. Our intelligent feedback system accelerates recovery and ensures proper form during physiotherapy exercises.

## Features

- **Real-time Body Tracking**: Advanced technology monitors your movements and provides instant feedback
- **Precision Guidance**: Personalized corrections to ensure every exercise is performed correctly
- **Home Convenience**: Professional-grade physiotherapy guidance from your smartphone
- **Progress Tracking**: Detailed analytics to monitor your recovery journey

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- pnpm (package manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AKZYP/PhysAlign.git
   ```

2. Navigate to the project directory:
   ```bash
   cd PhysAlign
   ```

3. Install dependencies:
   ```bash
   pnpm install
   ```

### Development

To run the development server:

```bash
pnpm dev
```

The application will be available at http://localhost:3000

### Building for Production

To create a production build:

```bash
pnpm build
```

To start the production server:

```bash
pnpm start
```

## Technology Stack

- **Frontend**: Next.js 16, React 19, TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, shadcn/ui
- **Validation**: Zod
- **Forms**: React Hook Form
- **Charts**: Recharts
- **Notifications**: Sonner
- **Deployment**: Vercel
